# 📚 VPL Library Directory

All libraries are located in `.vplib/` and are automatically managed by the VPL compiler.

## 🎨 Graphical User Interface (GUI)
- **`gui_core`**: Base functions for windows, clearing screens, and closing app.
- **`gui_widgets`**: High-level components (Labels, Panels, Professional Buttons).
- **`gui_theme`**: Professional Dark Mode color palette and theme management.
- **`gui_input`**: Real-time interactive Text Input fields with cursor and focus.
- **`gui_listbox`**: Advanced lists for files or data with selection highlighting.
- **`gui_events`**: Helper logic for hit-testing and event filtering.
- **`gui_layout`**: Grid and stack helpers for responsive element placement.

## ⌨️ Terminal User Interface (TUI)
- **`tui_widgets`**: Modern TUI components like Progress Bars and Status Lines.

## 🔐 Cryptography & Security
- **`crypto_pro`**: XOR Encryption with keys and string hashing algorithms.

## 🔢 Math Extensions
- **`math_prime`**: Prime number verification logic.
- **`math_fib`**: Fibonacci sequence generators.
- **`math_stats`**: Statistical helpers (Averages, sums, etc.).
